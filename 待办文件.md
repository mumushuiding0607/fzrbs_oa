# Finance 模块功能优化待办

> 计划陆续实现的功能优化清单

---

## 一、预算模块（budget）

### 高优先级
- [ ] **全局搜索** — 跨模块搜索：项目/合同/发票/订单编号
- [ ] **操作日志审计** — 关键数据修改留痕，支持按人员/时间查询
- [ ] **批量决算** — 支持批量选择多个项目一次性提交决算审批

### 中优先级
- [ ] **预算执行预警** — 当预算支出超过80%/90%/100%时自动提醒负责人
- [ ] **跨项目资金调拨** — 支持将一个项目的预算余额调拨至另一个项目
- [ ] **项目延期预警** — 根据合同期限自动提醒即将到期或已超期的项目
- [ ] **预算版本管理** — 支持预算变更历史记录，类似于合同变更

## 二、合同模块（contract）

### 高优先级

- [ ] **付款条件自动提醒** — 到期前N天自动提醒相关人员跟进回款

### 中优先级
- [ ] **合同到期预警** — 按月/季/年统计即将到期的合同数量
- [ ] **合同履行监控** — 设置里程碑节点，自动追踪项目进度与合同履约

### 低优先级
- [ ] **电子合同集成** — 对接电子签章系统实现在线签约
- [ ] **供应商合同管理** — 除客户合同外，增加采购/供应商合同管理
