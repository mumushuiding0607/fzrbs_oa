# Finance 模块功能优化待办

> 计划陆续实现的功能优化清单

---

## 一、预算模块（budget）

### 高优先级
- [ ] **全局搜索** — 跨模块搜索：项目/合同/发票/订单编号
- [ ] **操作日志审计** — 关键数据修改留痕，支持按人员/时间查询
- [ ] **批量决算** — 支持批量选择多个项目一次性提交决算审批

### 中优先级
- [ ] **预算执行预警** — 当预算支出超过80%/90%/100%时自动提醒负责人
- [ ] **跨项目资金调拨** — 支持将一个项目的预算余额调拨至另一个项目
- [ ] **项目延期预警** — 根据合同期限自动提醒即将到期或已超期的项目
- [ ] **预算版本管理** — 支持预算变更历史记录，类似于合同变更



---

## 二、合同模块（contract）

### 高优先级

- [ ] **付款条件自动提醒** — 到期前N天自动提醒相关人员跟进回款
- [ ] **合同变更记录** — 记录合同金额、期限、付款条件的每一次变更

### 中优先级
- [ ] **合同到期预警** — 按月/季/年统计即将到期的合同数量
- [ ] **合同履行监控** — 设置里程碑节点，自动追踪项目进度与合同履约

### 低优先级
- [ ] **电子合同集成** — 对接电子签章系统实现在线签约
- [ ] **供应商合同管理** — 除客户合同外，增加采购/供应商合同管理

---

## 三、发票模块（invoice）



### 中优先级
- [ ] **进项销项抵扣分析** — 增值税抵扣分析，税负计算
- [ ] **红字发票申请流程** — 线上申请、审批、作废冲红
- [ ] **批量操作** — 批量审批、批量导出、批量打印

### 低优先级
- [ ] **发票真伪查验** — 集成税务局发票查验API
- [ ] **电子发票PDF自动归集** — 开票后自动生成并发送给客户邮箱

---

## 四、广告订单模块（order）

### 高优先级

### 中优先级
- [ ] **组合套餐销售** — 支持将多个版面/时段打包成套餐销售
- [ ] **优惠折扣规则引擎** — 根据客户等级/订单金额自动应用折扣
- [ ] **广告撤刊/改刊流程** — 标准化撤刊申请和费用计算

### 低优先级
- [ ] **广告效果追踪** — 上刊后关联广告监播/回款数据
- [ ] **订单自动续约提醒** — 根据合同期限自动提醒客户续约

---

## 五、用印模块（useseal）

### 高优先级
- [ ] **用印与合同绑定** — 必须关联合同才能申请用印（部分场景）
- [ ] **用印文件自动归档** — 用印审批通过后自动上传至对应项目/合同

### 中优先级
- [ ] **用印次数统计** — 按部门/人员统计用印频次
- [ ] **用印类型分级** — 区分公章/法人章/合同章等，不同类型不同审批流

### 低优先级
- [ ] **电子用印集成** — 对接电子签章系统实现在线盖章

---

## 六、考勤模块（attandance）

### 高优先级
- [ ] **完整考勤报表** — 出勤天数/迟到/早退/请假/加班统计
- [ ] **请假销假闭环** — 请假申请→审批→销假确认

### 中优先级
- [ ] **加班调休管理** — 加班时长累计、转调休、过期提醒
- [ ] **排班管理** — 支持轮班/倒班等特殊排班

### 低优先级
- [ ] **考勤数据推送** — 推送至工资系统参与薪资计算
- [ ] **异常打卡AI审核** — 疑似代打卡等异常行为标记提示

---

## 七、流程模板模块（Flowtemplate）

### 中优先级
- [ ] **加签/转交** — 审批人可临时加签或转交他人
- [ ] **委托代理** — 长期出差等场景可设置流程代理

### 低优先级
- [ ] **可视化流程设计器** — 拖拽式设计审批流程，节点可配置
- [ ] **流程版本管理** — 流程变更有版本记录，可回滚

---

## 八、跨模块通用功能

### 高优先级
- [ ] **全局搜索** — 跨模块搜索：项目/合同/发票/订单编号
- [ ] **消息中心** — 统一站内通知：审批到达、超时、到期提醒
- [ ] **操作日志审计** — 关键数据修改留痕，支持按人员/时间查询

### 中优先级
- [ ] **数据驾驶舱** — 管理层汇总看板：收入/支出/回款/开票一览
- [ ] **批量操作** — 批量审批、批量导出、批量打印
- [ ] **批量操作** — 批量审批、批量导出、批量打印

### 低优先级
- [ ] **移动端适配** — 审批流移动端优化，支持手写签名
- [ ] **数据导入** — Excel批量导入项目/合同/发票数据
- [ ] **打印模板自定义** — 用户可调整打印格式，不改代码

---

## 实施优先级参考

| 优先级 | 功能 |
|--------|------|
| **P0**（立即开始） | 全局搜索、消息中心、操作日志审计 |
| **P1**（近期完成） | 合同收款进度跟踪、发票自动归档、预算执行预警、流程催办 |
| **P2**（规划中） | 批量决算/批量操作、付款条件自动提醒、合同变更记录 |
| **P3**（远期规划） | 电子签章集成、可视化流程设计器、数据驾驶舱 |
